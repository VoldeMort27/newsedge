Number Guessing game

This Python project involves creating a number guessing game where the user has to guess a randomly generated number between a specified range. The game handles errors such as invalid user input and provides feedback to the user to guess higher or lower. The code starts with importing the random integer function and setting up the lower and higher range values. An infinite while loop is then created to handle user input and try-except block handles any errors. The program checks if the user's guess is higher or lower than the generated random number and provides feedback to the user. Once the user guesses the correct number, the program breaks out of the while loop and ends. Additional complexity can be added to the game such as a limit on the number of guesses a user can make.
